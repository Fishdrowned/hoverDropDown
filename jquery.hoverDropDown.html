jQuery Plugin: hoverDropDown

Version: 1.0

2013-01-18T13:23UTC+08:00

Download

Example:

$("#container .handler").hoverDropDown($("#container .handler .dropdown"));

You can also define slide down duration, slide up duration and slide up delay:

$("#container .handler").hoverDropDown($("#container .handler .dropdown", { slideDown: 500, /* default 300 */ slideUp: "fast", /* default 300 */ slideUpDelay: 100 /* default 300 */ }));
